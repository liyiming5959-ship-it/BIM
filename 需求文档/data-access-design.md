# 深圳市BIM数据与能力中台 - 数据接入页面设计说明

## 一、页面基础信息

- **平台名称**：深圳市BIM数据与能力中台
- **所属模块**：数据中台
- **所属页面**：数据接入
- **面向用户场景**：
  - 系统管理
  - 实施服务
  - 运营

## 二、页面定位

该页面是市级BIM平台中"数据中台"的核心功能页面之一，主要用于对多类型BIM相关数据进行统一接入、汇集、归集、管理和查看，为后续的数据处理、数据解析、数据资产管理以及平台能力应用提供数据基础。

页面体现"市级平台、政府平台、中台系统"的产品定位，突出规范性、平台性、统一管理性和科技感。

## 三、核心业务需求

数据接入页面主要用于实现以下内容：

1. 接入SZ-IFC模型数据及源模型数据等BIM数据
2. 接入图纸数据、倾斜摄影数据、项目资料文件等相关数据
3. 接入IoT数据
4. 为市级BIM平台（1.0）的能力及应用提供统一的数据基础
5. 支持BIM模型和对应业务数据的整体采集和存储
6. 支持完善模型参数及属性信息
7. 支持对不同来源、不同类型的数据进行归集、管理和查询

## 六、页面布局设计

整体按照政府性质的市级平台首页、落地页形式布局：

1. **顶部标题及简介区**：包含页面名称、简介、状态标签和操作按钮
2. **数据概览区**：采用网格布局，展示各类数据接入统计
3. **功能入口区**：以卡片形式展示4个子菜单入口
4. **流程/说明区**：可视化展示数据接入流程
5. **动态记录区**：表格形式展示近期接入动态

## 七、UI风格设计

整体采用"政务+科技"风格：

- **简约**：界面简洁明了，无多余装饰
- **大方**：布局规整，视觉层次清晰
- **专业**：体现政府平台的专业性和权威性
- **科技感**：适当运用现代UI元素和动效
- **色彩方案**：以蓝、白、灰为主
  - 主色：#0052D9（深蓝色）
  - 辅助色：#00B4D8（亮蓝色）
  - 背景色：#F3F4F6（浅灰色）
  - 卡片背景：#FFFFFF（白色）
- **卡片风格**：圆角矩形，轻微阴影，边框为浅灰色
- **图标风格**：Font Awesome图标库，风格统一
- **按钮风格**：圆角矩形，主按钮为蓝色，次要按钮为白色边框

## 八、设计细节

1. **响应式设计**：适配不同屏幕尺寸，在小屏幕设备上自动调整布局
2. **交互反馈**：卡片和按钮有悬停效果，提升用户体验
3. **视觉层次**：通过卡片、阴影、颜色等元素创建清晰的视觉层次
4. **业务识别度**：使用与BIM模型、图纸、IoT、数据流转相关的视觉元素或图标
5. **可扩展性**：页面结构设计便于后续增加子功能页面

## 九、技术实现

- **前端框架**：HTML5 + Tailwind CSS
- **图标库**：Font Awesome 4.7.0
- **图表库**：Chart.js（预留）
- **布局方式**：Flexbox + Grid
- **交互逻辑**：原生JavaScript
